# MediNLP - House MD Dialogue Analysis

## Project Description
This project was developed for the Natural Language Processing course.

The aim of this project is to analyze medical dialogues from the House MD dataset using NLP techniques.

## Features
- Text preprocessing
- Intent classification
- Emotion analysis
- TF-IDF vectorization
- Logistic Regression model
- Medical dialogue prediction

## Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Google Colab

## Results
Intent classification accuracy reached approximately 31%.
